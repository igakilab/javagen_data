### WorkA8実行例(１）

- 全ての実行例において、文字列や数字の横に <- とあるところはユーザーによる入力を表しています。

```sh
 $ java WorkA8
 整数を1つ入力してください
 100　<-
 HP=100
 HP= 90
 HP= 80
 $
```
