### 実行例

- テキストを入力する部分はユーザーによる入力を表し、実際のテキストは表示のみ行います。

```sh
$ java WorkF5
テキスト１を入力してください：
Hello World! <-
テキスト２を入力してください：
I love Java! <-
テキスト３を入力してください：
Programming is fun! <-
テキスト４を入力してください：
Java is awesome! <-
テキスト１：Hello World!
テキスト２：I love Java!
テキスト３：Programming is fun!
テキスト４：Java is awesome!

$
```
